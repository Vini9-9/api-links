## CRUD na API:
* Criar: http://localhost:3000/?action=c&name=nome&url=url
* Ler: http://localhost:3000/
* Atualizar: http://localhost:3000/?action=u&name=nome&url=url
* Deletar: http://localhost:3000/?action=d&name=nome&url=url
